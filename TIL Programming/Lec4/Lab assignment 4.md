## 1.2. Calculats their sum


```python
# YOUR CODE HERE
file_path = 'addition.txt'
summ = int()
list1 =[]
with open(file_path,'r') as fp:
    lines = fp.readlines()
    for line in lines:
        line = line.strip()
        list1.append(line)
    while '' in list1:
        list1.remove('')
    list1 = list(map(int,list1))
    for line in list1:
        summ = summ + line
    print(summ)
        
```
